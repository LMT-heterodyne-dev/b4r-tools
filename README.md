# b4r-tools
:rocket:Tools for B4R/LMT data reduction.

**The scripts are still under development. We are not responsible for the outputs now.**

**************************************************************
Python environment
**************************************************************

```terminal
$ cd b4r-tools
$ pip install pipenv
$ pipenv install
$ pipenv --venv
```

You need to change parameters in b4rtools. Then,
```terminal
$ ./b4rtools
```
