</span>

**************************************************************
Python environment
**************************************************************

```terminal
$ cd b4r-tools
$ pip install pipenv
$ pipenv install
$ pipenv -venv
```

You need to change parameters in b4rtools. Then,
```terminal
$ ./b4rtools
```
